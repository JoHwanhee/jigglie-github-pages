Jiggle - 간편한 댓글 서비스

Jiggle은 웹사이트에 쉽게 통합할 수 있는 댓글 서비스입니다. Jiggle을 사용하면, 여러분의 웹페이지에 방문자가 자유롭게 댓글을 남길 수 있습니다.

사용자는 아래와 같이 간단한 스크립트를 추가하여 Jiggle 서비스를 자신의 웹사이트에 추가할 수 있습니다. 스크립트에서 domain 속성에는 자신의 도메인 이름을 입력해야 합니다.

<script src="https://3o7kob0fb5.execute-api.ap-northeast-2.amazonaws.com/prod/embed.js"
    domain="yourdomain.com"></script>

Jiggle 시작하기

  1. 위의 스크립트를 복사합니다.
  2. 자신의 웹사이트 <body> 태그 안에 붙여넣습니다.
  3. domain 속성에 자신의 도메인을 입력합니다 (예: mywebsite.com).
  4. 페이지를 저장하고, 브라우저에서 새로 고침하여 Jiggle 댓글 서비스가 제대로 작동하는지 확인하세요.

Jiggle이 올바르게 설치되면 방문자들은 댓글을 남길 수 있는 인터페이스를 보게 됩니다.

예시 코드

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Jiggle - 내 웹사이트에 댓글 추가</title>
</head>
<body>
  <script src="https://3o7kob0fb5.execute-api.ap-northeast-2.amazonaws.com/prod/embed.js"
    domain="yourdomain.com"></script>
</body>
</html>
